React-bootstrap-table

API & Attributes

The attributes in <BootstrapTable>:

Use data (required) to specify the data that you want to display on table.
Use height to set the table height, default is 100%.
Use striped to set table be a striped columns. Like bootstrap table class .table-striped.
Use hover to enable table hover. Like bootstrap table class .table-hover.
Use condensed to set a condensed table. Like bootstrap table class .table-condensed. Use pagination to enable pagination on table.
Use insertRow to enable row insertion on table.
Use deleteRow to enable row deletion on table. If true, you must use selectRow to enable row selection.
Use columnFilter to enable the column filter on table.
Use search to enable the table search.
Use selectRow to enable row selection on table. This attribute accept an object,
This object may contain these properties:
    mode: radio/checkbox (required), to specify the single selection or multi selection in table.
    clickToSelect: true/false (optional), if true, click the row will trigger selection on that row, default is false
    clickToSelectAndEditCell: true/false (optional), if true, click the row will trigger selection on that row, and also trigger cell editing if you enabled cell edit.
    bgColor: css color (optional), if assigned, this color will present on backgroud if row selection.
    selected: an array object (optional), it's for default selected row on table, give an array object which contain selected row keys.
    onSelect: callback function (optional), accept a callback function, if a row be selected, this function will be called.
    onSelectAll callback function (optional), accept a callback function, if select all in checkbox mode, this function will be

Use cellEdit to enable the cell editing on table
cellEdit accept a object which contain these properties in blow:
    mode: click/dbclick (required), to spectify which condition will trigger cell editing.
    blurToSave: true/false (optional) if true, when mouse blur on input field will trigger a save on cell, default is false.
    afterSaveCell: callback function (optional), accept a callback function, after save cell, this function will be called.

Use options to set other options setting for table
In newest version, the options accept a object and include these properties as follow:
    sortName: column name (optional), to spectify which column should be the default sorting column.
    sortOrder: asc/desc (optional), to spectify the order of default sorting column.
    afterTableComplete: callback function (optional), A hook for after table render complete.

The attributes in <TableHeaderColumn>:

Use isKey (required) to specidy which column is unique.
Use width to set the width on column.
Use dataField (required) to specidy which column you want to show on this column.
Use dataAlign to set align in column. Available value is left, center, right, start and end.
Use dataSort to enable the sorting in column. Default value is false(disabled).
Use dataFormat to customize this column.Must give it as a function.
Use hidden to enable column hidden
Use editable to specify column editable, default is true. This attribute only works on cellEdit be setted in <BootstrapTable> certainly.

About TableDataSet

After v0.8.0, rect-bootstrap-table provide the ability of updating table data on fly. How to do it?

1. Import TableDataSet
var ReactBsTable  = require('react-bootstrap-table');
var TableDataSet = ReactBsTable.TableDataSet; //import TableDataSet
2. Create a TableDataSet object with you data. 
var dataSet = new TableDataSet(products);
3. Pass this object to data 
<BootstrapTable data={dataSet}>
   ...
</BootstrapTable>
4. If you want to reset or update table data in any time, you can: 
dataSet.setData(newproducts);
